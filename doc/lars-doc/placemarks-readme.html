placemarks.py:

The placemarks.py script is run by calling python placemarks.py <path of data.txt file> <directory to relevant images> <flags>. It will produce a file "out.kml" in the directory it was called from that can be opened in google earth.

Flags:
-r : if you want the paths to the images in the kml file to be relative
-w : if you want the kml file to be openable in windows
