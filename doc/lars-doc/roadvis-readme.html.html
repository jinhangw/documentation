Roadvis REAMDE

The roadvis.py script takes one argument, the data file which should be in the same directory as all of the images that it references. roadvis.py requires pygame and python2.6 to be run. pygame instruction installations can by found at www.pygame.org. The map-data/ folder inside the roadvis directory needs to contain the images to be used as maps. The coords file contains the location information for evey image in the following example format for the image sc1.png with top right and bottom left coordinates specified following the colon and separated by "||": sc1.png: 40,29,04.79 N; 80,02,11.09 W || 40,24,14.29 N; 79,50,44.35 W.
