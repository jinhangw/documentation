How to measure distance/ accuracy of two point clouds

1. Load the two point cloud layers.

2. Press Ctrl key on the keyboard, and select both layers. Now you need to cut the two layers into exact same size. Click on ’segment’ tool from the tool bar. Click to set the start point, click again to set another point. When you finish setting the boundary, right click to release the cursor. In the segment menu bar, choose to keep the inner part of the polygon and accept the changes by clicking .

3. Press Ctrl key to select the new segmented layers. On the tool bar, click the button for ‘compare cloud/cloud distance’. In the pompt-up window, see if the parameters on each tab are proper.

4. On the third tab, click Compute to get the result.

5. You can view the histogram by clicking the ‘Show histogram’ button in the tool bar.

6. Save your segmented layers.


How to generate the plane for accuracy and completeness

create a plane in blender

1. How to generate a point cloud of plane in Blender

First, generate a mesh of a plane. In object editing mode, subdivide the plane as many times as possible (depends on the PC system capacity). Save the file as .ply ASCII file.

Open the .ply ASCII file of the mesh generated in text editor. Delete all information relating to ‘face’ (scroll down to find the start. It has different number of columns than points; and each row starts with a number ‘4’.)


reconstruct the table surface(plane) in VisualSFM. Clean up the point cloud in Meshlab.

2.(How to measure and scale point cloud)

In CloudCompare, load the cleaned up table surface. Use from the toolbar, select the mode to measure the distance of two points. The short side of the table has a measurement of 1.37(Dtarget). The actual distance is 0.915m (Dactual). So the scaling factor Sf=Dactual/Dtarget =0.668. In the toolbar, go to Edit > Multiply/scale: in the fx, fy and fz fields, enter the Sf factor (same for all dimensions).

3. Align the two planes.

Load the blender plane. Select both the blender plane and the table layer. Align the blender plane to the table surface. i.e. the reference is table surface cloud. Check the box to allow ‘adjust scale’.

Select both layer to compute cloud-to-cloud distance. Set the blender plane as reference.

Use to show more statistical results.


More info: http://www.cloudcompare.org/doc/wiki/index.php?title=How_to_compare_two_3D_entities


4. Chop (to create exact same shape) to better align the two planes.

After adjusting the scale and aligning the two planes, we can chop the table layer to better align the two planes. First, press ctrl to select both the blender plane and the table layer. Use the Segment tool from toolbar to chop both plane together into the exact same shape. Then, repeat the aligning process. Since they are in exact same shape, we can find the exact matching anchor points in both plane. The alignment should be more accurate then.
