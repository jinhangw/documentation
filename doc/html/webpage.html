Webpages for Road Investigation project


The website contains project related information. It contains manuals and instructions on building and operating the system, as well as code and scripts on algorithm and data training.

Bootstrap:

The main css style is from Bootstrap. A few jquery files (in folder js) and css files (in folder css) were added.
Refer to Bootstrap instructions when modifying the layout.

===========================CMU AFS=================================

Setup:

Email SCS help to apply for project space. Create the 25GB space on afs server.

For later on settings, follow the instruction here: http://www.cs.cmu.edu/~help/web_publishing/web_publishing_intro.html

Every time log in:

In a terminal, type:

ssh user@unix.andrew.cmu.edu

ls (list the files in your personal afs folder)

aklog cs.cmu.edu (do this if your account is not a cs account, and you want to access this cs directory)

cd /afs/cs.cmu.edu/project/road (go to

cd www (all website materials should be stored here)

Copy files into project/road/www:

If you have a SCS account….

(I don’t know how to do it)

If you do not have a SCS account, you have to upload the local file to your afs space first. Then, copy the file from your afs space to the project space. For example:

In a new terminal (on local disk), type:

scp ~/index.html user@unix.andrew.cmu.edu: (this is to copy the file to your afs space)

scp ~/folder/* user@unix.andrew.cmu.edu: (this is to upload all files in this folder)

ssh user@unix.andrew.cmu.edu (log into your afs space)

ls (list the files to double check that the file is copied)

aklog cs.cmu.edu

mv ~/index.html /afs/cs.cmu.edu/project/road/www (this is to move the file from your space to the afs project space).

Exit:

In the terminal, type:

exit

to exit the afs server.
