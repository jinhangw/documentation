First Time Android Setup

Under settings, connect to a wifi network. Still in settings, navigate to Security and under the Device Administration section select Unknown Sources in order to install our own app. Now launch the google play app store and sign into a google account (preferably cmupothole@gmail.com). Install the Dropbox app from within the App store. Sign into the cmupothole@gmail.com dropbox account and navigate to the /Android-Apps/ folder where you can install the necessary app.
The phone is now set up.
